## 下载安装开发工具

[Visual Studio Code](https://code.visualstudio.com/)

## 配置 prettier

安装 Prettier - Code formatter 插件，配置 settings.json

```json
{
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.tabSize": 2
}
```

## 下载项目

```git
git clone git@gitee.com:MingMingBuZai/vitepress.git
```

## 安装依赖

```
pnpm install
```

## 运行项目

```
npm run dev
```
